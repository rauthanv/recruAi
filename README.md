RecruAi

RecruAi is a simple AI-powered platform for creating and conducting technical interviews. It allows users to generate interview questions, schedule sessions, run voice-based interviews, and review feedback.

Features

AI-generated interview questions

Shareable interview links

Voice interview sessions using Vapi.ai

Supabase authentication

View completed interviews and feedback

Clean dashboard for managing all interviews

Tech Stack

Next.js 15

React

Tailwind CSS & ShadCN UI

Supabase

Vapi.ai
